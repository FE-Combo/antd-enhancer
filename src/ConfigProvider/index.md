# ConfigProvider
- 新特性
  - 为 antd-enhancer 提供统一的全局化配置。

# 基本参数

| 参数  | 说明 | 类型     | 默认值 |
| ----- | ---- | -------- | ------ |
| localeEnhanced | 语言包，语言包可到 antd-enhancer/lib/ConfigProvider/locales 目录下寻找 | `object` | `en`  |

[ConfigProvider配置参考antd文档](https://ant.design/components/config-provider-cn/#API)
